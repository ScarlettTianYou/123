# Action-Official


Build OpenWrt firmware OpenWrt(https://github.com/openwrt/openwrt) using GitHub Actions  

Very simple firmware, only with passwall.

用官网openwrt/openwrt加上passwall，云编译，简洁小巧。


For now only x86_64 works.

目前只试了x86_64的。
